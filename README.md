# AI Utility Skills

A curated collection of practical AI-powered utility skills designed for everyday productivity and automation.

## 🚀 Features

- **Audio Transcribe** - Convert video/audio files to timestamped text with online (Groq) and offline (Whisper) options
- **Mind Map Generator** - (Coming soon) Generate visual mind maps from text and ideas
- More practical skills coming soon...

## 📁 Skills Structure

Each skill is organized as an independent module with:
- `SKILL.md` - Complete documentation and usage guide
- `scripts/` - Implementation files
- `pyproject.toml` - Dependencies and configuration

## 🛠️ Getting Started

Each skill can be used independently. Navigate to the skill directory and follow its setup instructions.
